# my_FileTransfer_1   基于TCP/UDP的文件传输系统
**Jackkii 20201216**

---
可实现同一台电脑中Win10系统内、Linux系统内、Win10系统与Linux系统的文件传输通信
---
### 运行环境：
1. Win10系统 / Ubuntu20.04
2. Qt5.6.1 / Qt 5.9.9

---------------------**V1.0版本**---------------------
- 功能：实现基于tcp协议的文件传输系统
